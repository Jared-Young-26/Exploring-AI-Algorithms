# Artificial Intelligence Algorithms — Fall 2025

This repository contains a collection of algorithm-focused programming projects completed for an Artificial Intelligence course during the Fall 2025 semester. The projects emphasize implementing core AI and machine learning techniques **from first principles**, with minimal reliance on high-level libraries.

Rather than treating algorithms as black boxes, each project explores the underlying mathematics, data flow, and design tradeoffs involved in classical AI systems.

---

## Project Focus

The primary objective of this repository is to demonstrate a working understanding of fundamental AI algorithms through hands-on implementation, analysis, and experimentation.

The projects include:

### Neural Networks (From Scratch)

* Manual implementation of a feed-forward neural network
* Custom activation functions
* Forward and backward propagation
* Weight initialization and training dynamics
* Evaluation of learning behavior and convergence

### Hidden Markov Models (HMM)

* State transition and emission probabilities
* Forward and backward algorithms
* Sequence likelihood computation
* Probabilistic modeling of temporal data

### K-Means Clustering

* Unsupervised clustering using distance-based methods
* Centroid initialization and convergence
* Cluster assignment and updating
* Analysis of clustering behavior and limitations

### Regression Analysis

* Linear and polynomial regression models
* Parameter estimation and error minimization
* Model evaluation and interpretation
* Comparison of regression approaches across datasets

---

## Learning Objectives

These projects were designed to reinforce:

* Core AI algorithmic concepts
* Mathematical foundations of machine learning
* Probabilistic reasoning and inference
* Model evaluation and limitations
* Algorithmic tradeoffs in performance and accuracy
* Debugging and validating numerical methods
